# Autonomous Mobile Robot Navigation Using A* Algorithm

## Project Objective

To develop an Autonomous Mobile Robot capable of mapping its environment, finding the shortest path using the A* algorithm, and avoiding dynamic obstacles.

## Tools Used

* Python
* NumPy
* Matplotlib
* Google Colab

## Features

* Occupancy Grid Mapping
* A* Path Planning
* Dynamic Obstacle Avoidance
* Path Replanning

## Results

The robot successfully mapped the environment, generated the shortest path, detected obstacles, and replanned its route.
